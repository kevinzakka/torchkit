torchkit.utils.torch_utils¶

Pytorch-related utils.

torchkit.utils.torch_utils.freeze_model(model, bn_freeze_affine=False, bn_use_running_stats=False)[source]¶

Freeze PyTorch model weights.

Parameters

  • model (Module) – The model to freeze, a subclass of torch.nn.Module.

  • bn_freeze_affine (bool) – If True, freezes batch norm params gamma and beta.

  • bn_use_running_stats (bool) – If True, switches from batch statistics to running mean and std. This is recommended for very small batch sizes.

Return type

None

torchkit.utils.torch_utils.get_total_params(model, trainable=True)[source]¶

Get the total number of parameters in a PyTorch model, from 1.

Example usage:

class SimpleMLP(nn.Module):
    def __init__(self):
        super().__init__()
        self.fc1 = nn.Linear(3, 16)
        self.fc2 = nn.Linear(16, 2)

    def forward(self, x):
        out = F.relu(self.fc1(x))
        return self.fc2(out)

net = SimpleMLP()
num_params = torch_utils.get_total_params(net)

# prints the following:
+------------+------------+
|  Modules   | Parameters |
+------------+------------+
| fc1.weight |     48     |
|  fc1.bias  |     16     |
| fc2.weight |     32     |
|  fc2.bias  |     2      |
+------------+------------+
Total Trainable Params: 98
Parameters

  • model (Module) – The model, a subclass of torch.nn.Module.

  • trainable (bool) – Only return trainable parameters.

Return type

int

Returns

Either all model parameters or solely the trainable parameters.
